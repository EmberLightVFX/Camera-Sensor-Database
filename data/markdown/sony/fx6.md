# Sony - FX6

## Info

Sony only provides the native sensor dimensions and there is no data available for the exact sensor dimensions for cropped recording modes (i.e. S&Q mode over 60fps). Sensor entry will be updated if/when Sony releases this information.

The provided native sensor dimensions result in a 3:2 aspect ratio, but shooting Open Gate is not possible on the FX6, so currently 17:9 is the recording format that uses the most area of the camera's native sensor.
